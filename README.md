# Hardware-Aware AI Optimization

Welcome to my portfolio. As the CEO of Nota AI, I focus on bridging the gap between high-performance AI models and resource-constrained edge devices.

## 🚀 Featured Project: Ultra-Lightweight GenAI for Edge Devices

This project demonstrates a specialized framework for optimizing Generative AI (LLMs) and Computer Vision models using hardware-aware techniques. It focuses on reducing latency and power consumption without compromising accuracy.

### 🛠 Key Features
- **Automated Model Compression:** Pruning and quantization tailored for specific NPU/GPU architectures.
- **CV to GenAI Pipeline:** Seamless transition techniques for deploying vision-language models on-device.
- **Latency Estimation:** Real-time feedback loops to predict model performance on target hardware during the training phase.

- **Frameworks:** PyTorch, ONNX, TensorRT
- **Expertise:** Model Pruning, Quantization, Knowledge Distillation
